My apologies for not sending this out earlier in the month but I've been contaminated with some awful virus or plague or some deadly manflu.

This month I've persuaded Joshua Hesketh to give us the talk he recently gave at PyconAU about the Zuul project gating system. Zuul is a program that is used to gate the source code repository of a project so that changes are only merged if they pass tests.

It's a good follow-up to the Gerrit talk that came along last month. So I hope you brave the weather and meet us up at Hotel SoHo for a drink and a schnitzel and some FOSS development discussions!
