Grid Computing:
This is about building a huge, worldwide compute infrastructure,

mainly for the upcoming experiments
of the Large Hadron Collider at CERN / Geneva. These
will produce about 10 petabyte of data per year, which
are being accessed by 6000 physicists worldwide.
The current Grid infrastructure, in the context of
the European Union initiative EGEE, comprises about
14,000 CPUs, all running under Linux.


The ROOT toolkit:
This is a graphical analysis toolkit comprising several
million lines of code. As it is based around a C++
_interpreter_, it can be used for virtually any purpose.
Just link your own libraries with the interpreter and
access the functions from you C++ scripts. ROOT is
one of the largest open source projects (surely after
Apache and the Linux kernel, but still …), but still
mostly unknown outside the physics community. I perceive
the benefit of ROOT to the OSS community to be potentially
large, so I would like to do some dissemination work
for this project.
