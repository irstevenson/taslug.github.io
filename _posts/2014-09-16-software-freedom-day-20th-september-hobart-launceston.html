WHEN:</h1>
Saturday 20th September, 10am to 3pm

WHERE:</h1>
Launceston: Launceston Bowls Club, Home Point Parade, Royal Park Launceston (map</a>)

Hobart: Lenah Valley Community Hall, Creek Road, Lenah Valley (map</a>)

WHAT:</h1>
Software Freedom Day is a grassroots celebration of Open Source Software happening in conjunction with others around the world. We hope to raise awareness of Open Source Software by showing off different versions of Linux, business software such as word processors and accounding packages, graphical applications and games and portable applications that can be used from a USB stick. There will be giveaways.

In Hobart there will be a BBQ, some PCs for games and ongoing demonstrations of loading modified operating systems on phones and tablets, and of educational software for the Raspberry Pi.

In Launceston, there will be some game developers on site working on making a game. They will be happy to talk to the general public about what they are doing and how to go about the process of creating a computer game, along with the tools used for design and collaboration.
