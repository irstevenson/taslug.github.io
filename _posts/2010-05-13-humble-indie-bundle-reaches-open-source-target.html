For anybody who wasn't aware, Wolfire Games has been running a "Pay what you want" sale for a suite of five independant cross platform (Windows, MacOS and Linux) games. When making the purchase, contributors are able to choose how their money is split up between the game developers and two not for profit charities.

On the 10th of May, it was announced via Twitter</a> that if the week long promotion reached USD$1,000,000, three of the games would release their source code.

That milestone has been reached, the offer has been extended for a few more days, and a fourth game within the suite is releasing source.

More information can be found here:
http://www.wolfire.com/humble</a>

An interesting statistic: on average, Linux users on average donate larger figures than MacOS or Windows users and have contributed a quarter of the money raised.
